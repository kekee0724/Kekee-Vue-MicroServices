# wx_login

#### 介绍
springboot整合微信小程序

#### 软件架构
项目描述：在微信小程序中通过与Springboot操作数据库实现登录验证，其中我是用springboot整合mybatis-plus 和mysql操作数据库

主要技术：SpringBoot、lombok、mybatis-plus、mysql 、微信小程序
